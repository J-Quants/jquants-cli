# Category: idx (Indices) — Command Reference

## idx daily-topix — TOPIX指数四本値

```sh
jquants idx daily-topix
jquants idx daily-topix --from 2021-09-01 --to 2021-09-07
```

## idx daily — 指数四本値

```sh
jquants idx daily --code 0028
jquants idx daily --date 2021-09-07
jquants idx daily --code 0028 --from 2021-09-01 --to 2021-09-07
```
